# Types - Number

## Floating Precision

```typescript
let x = 0.2 + 0.1; // 0.30000000000000004
let x = (0.2 * 10 + 0.1 * 10) / 10; // 0.3
```

## Adding Numbers and Strings

```typescript
let a = 10 + 20; // 30
let b = "10" + 20; // 1020
let c = 10 + "20"; // 1020
let c = "10" + "20"; // 1020
let d = 10 + 20 + "30"; // ???
```

> Hint: The JavaScript interpreter works from left to right.
