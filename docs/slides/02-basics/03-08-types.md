# Types - Object

```typescript
let lastName = "Muster";
let obj = {
  firstName: "Hans",
  lastName,
  hobbies: ["Tennis", "Lesen"],
};

let obj = {};
obj.firstName = "Hans";
obj.lastName = "Muster";
obj.hobbies = ["Tennis", "Lesen"];
```
